# Configurações

A tela reúne preferências do aplicativo sem misturá-las ao planejamento
financeiro:

- **Aparência** permite seguir o aparelho ou forçar os temas claro e escuro;
- **Ajuda** reabre a apresentação educativa e o tour da Home;
- **Sobre** apresenta produto, versão pública, release Android e a assinatura
  institucional.

A seção Sobre é somente informativa. Ela obtém os metadados por
`expo-constants`, usa a identidade centralizada em `src/shared/app-identity.ts`
e não possui navegação, toque ou persistência.

`Powered by Leahcim` identifica a marca responsável pelo produto. Essa
assinatura não declara a existência de uma entidade jurídica registrada.
